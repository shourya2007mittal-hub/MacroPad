# MicroPad Air1
<img width="793" height="651" alt="Screenshot 2026-06-04 025024" src="https://github.com/user-attachments/assets/a4cbdd11-cc0d-4eef-9c66-fa366a7ca67a" />
This is a Custom Xiao Rp2040  Based MicroPad , It has 8 Switches , a roter encode and an 0.91 inch  oled screen . 
We used matrix for switches so it uses less gpio pins and it has 3 layers the rotor key is used to change the layers
and the selected layer is  displaced on the screen.so in total we have  keys output 8x4=32 outputs.

## Features

- Based on Xiao Rp2040
- Has total 32 Outputs
-  Uses Keyboard Matrix

-  ## Firmware
- Firmware is based on QMK.
